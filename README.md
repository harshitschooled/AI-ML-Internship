# AI & Machine Learning Internship Portfolio

## Overview

This repository documents my learning journey and practical work completed during an AI & Machine Learning Internship. It serves as a structured portfolio of daily assignments, experiments, and implementations covering data preprocessing, exploratory data analysis, feature engineering, and machine learning fundamentals.

The primary objective of this repository is to demonstrate the progressive development of AI/ML skills through hands-on practice using industry-standard tools and workflows.
